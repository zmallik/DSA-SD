# Functional Requirements
## Core Requirements
1.  should be able to view a list of coding problems.
2. Users should be able to view a given problem, code a solution in multiple languages.
3. Users should be able to submit their solution and get instant feedback.
4. Users should be able to view a live leaderboard for competitions. <br/>

**Below the line (out of scope):**

 - User authentication
 - User profiles
 - Payment processing
 - User analytics
 - Social features

# Non-Functional Requirements
- availability >> consistency
- low latency submission processing (<5s)
- scale for competitions of 100k
- security and isolation of code runs
